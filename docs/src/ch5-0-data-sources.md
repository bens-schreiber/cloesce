# Data Sources

The [previous chapter](./ch4-0-models.md) covered how to define Models in Cloesce, including how to use [D1 databases](./ch4-1-d1-backed-model.md), [KV namespaces](./ch4-4-kv-fields.md), and [R2 buckets](./ch4-5-r2-fields.md) as data sources for your Models.

Before diving into APIs and Services, it's important to understand how data is actually retrieved and manipulated in Cloesce.

This chapter provides a reference for how to write Data Sources in Cloesce, which are the building blocks for all data retrieval in your application.
