# APIs and Services

This chapter covers how to define [APIs](./ch6-1-rest-apis.md) in Cloesce, typed REST specifications that are generated as backend stubs and frontend client methods, automatically routed by the Cloesce runtime.

Additionally, this chapter covers [Services](./ch6-6-services.md), which allow developers to write APIs that are not directly tied to a [Model](./ch4-0-models.md).
