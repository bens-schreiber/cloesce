# Models

In Cloesce, a _Model_ is an entity that exists across the entire project stack.

In this chapter, we will explore the various features of Cloesce Models, including

- defining [D1](https://developers.cloudflare.com/d1/), [KV](https://developers.cloudflare.com/kv/), and [R2](https://developers.cloudflare.com/r2/) backed properties
- relationships between Models
- generated CRUD methods
- API methods
